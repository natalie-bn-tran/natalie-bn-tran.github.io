Network Services Lab (Evan's Lab)

Background

Evan's Lab focuses on infrastructure lab and covers networking, setting up services, Active Directory, and using firewalls to secure a network. Evan created the lab for SWIFT Pals with a goal of helping students gain hands-on experience in cybersecurity.

Personal Experience

Original Documentation
I started the lab in March 2024 with little knowledge of networking and how services work. Three months later, I became the second person to finish the lab.

This is one of the most valuable learning experiences I've had. I was better able to understand how services interact with each other/Active Directory and strengthen my troubleshooting skills. It also teaches important lessons like patience and perseverance, especially when services are incorrectly set up and when installation processes need to be restarted.

OS Requirements

  • pfSense (Router)
  • Windows Server 22 (Domain Controller)
  • Windows Server 2016 (XAMPP Web Server)
  • Windows Server 2012 R2 (IIS Web Server, Secondary Domain Controller)
  • Windows 10 (Client)
  • Windows 7 (SMB)
  • Ubuntu 22.04 (MySQL)
  • Centos 7 (Apache Web Server)
  • Debian 11 (Nginx Web Proxy)
  • Ubuntu 18 (SFTP)

Service Requirements

  1. Active Directory
    1. Primary Domain Controller and Primary DNS Server → Windows Server 2022
    2. Secondary DC and DNS → Windows Server 2012 R2
    3. All machines should be joined to the Domain
  2. XAMPP Web Server
    1. WordPress Website using Ubuntu 22.04 for Database
  3. SMB
    1. Windows 7 shares proprietary SWIFT data (🍆) to only Windows 10 Client
  4. MySQL
    1. Database for Windows Server 2016 and Centos 7
    2. Ensure database is secure from Justin's hands.
  5. Apache Web Server
    1. WordPress Website using Ubuntu 22.04 for Database
  6. Nginx Web Proxy
    1. Proxy for IIS Website
  7. SFTP
    1. Share classified SWIFT HR (🍑) to only Windows 10 Client

Security Considerations

  • SWIFT shouldn't be leaking out proprietary data to other clubs. That is how insurrections occur.
  • SWIFT does not want violations of GDPR, as we operate worldwide. Make sure that HR data is locked down.
  • The IIS web server shouldn't be reachable from any of its computer's interfaces. That's what the Web Proxy is for!
  • NSA released a different beast in 2017, however, you are running the same animal. Please name it.
  • WordPress creds 🤔
  • We don't accept credit cards, but we still want to be 1/12th PCI-DSS compliant. Please follow PCI-DSS requirement 1 for all machines.
